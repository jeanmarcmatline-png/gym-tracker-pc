Guide de transfert OneDrive
PC → Téléphone (config)
1. Clique "Générer config_mobile.json"
2. Le fichier se télécharge dans Téléchargements
3. Déplace-le dans ton dossier OneDrive
4. Sur le téléphone, ouvre OneDrive
5. Télécharge le fichier
6. Dans l'app mobile : "Importer config"
Téléphone → PC (séance)
1. Dans l'app mobile : "Exporter séance"
2. Partage vers OneDrive
3. Sur le PC, ouvre OneDrive
4. Télécharge le fichier séance
5. Ici : "Sélectionner le fichier séance"
6. La séance s'enregistre dans gym.db
